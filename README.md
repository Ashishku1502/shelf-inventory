# Shelf Inventory System 📦

A modern, high-performance inventory management system designed for "Shelf OS". This application provides a comprehensive suite of tools for managing product batches, tracking stock levels, and handling approval workflows with a premium, reactive interface.

## ✨ Features

- **Dynamic Inventory Tracking**: Real-time monitoring of stock levels for packaged and loose items.
- **Batch Management**: Detailed tracking of UPC numbers, batches, prices, and weights.
- **Approval Workflow**: Integrated system for items sent for approval, with clear status indicators (Pending, Approved, Rejected).
- **Responsive Design**: A sleek, side-nav based layout optimized for productivity.
- **Intelligent Search**: Fast, client-side filtering for products and inventory items.
- **Modern UI**: Built with React and optimized for a premium user experience.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ashishku1502/shelf-inventory.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: Premium Inline Design System
- **State Management**: React Hooks (useState, useEffect)

## 📁 Project Structure

- `ShelfInventory.jsx`: Main application component containing the core logic and UI.
- `src/`: Additional source files and assets.
- `vite.config.js`: Configuration for the Vite build tool.
